Aqui está um exemplo de um link estilizado para um formulário de cadastro (*signup*) usando HTML e CSS:

### HTML:
```html
<a href="signup" class="signup-link">Cadastre-se</a>
```

### CSS:
```css
.signup-link {
    display: inline-block;
    text-decoration: none;
    background-color: #4CAF50; /* Cor de fundo */
    color: white; /* Cor do texto */
    padding: 10px 20px; /* Espaçamento interno */
    font-size: 16px; /* Tamanho da fonte */
    border-radius: 5px; /* Cantos arredondados */
    transition: background-color 0.3s ease; /* Transição suave */
}

.signup-link:hover {
    background-color: #45a049; /* Cor ao passar o mouse */
}
```

### Como funciona:
1. O botão/link é estilizado para se parecer mais com um botão do que com um link tradicional.
2. A classe `signup-link` aplica o estilo base, incluindo cores, tamanho e arredondamento dos cantos.
3. A pseudo-classe `:hover` altera a cor de fundo ao passar o mouse sobre o link, adicionando interatividade.

Você pode ajustar as cores, tamanhos e outros detalhes conforme necessário.
