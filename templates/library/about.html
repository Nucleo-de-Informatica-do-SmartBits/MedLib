O que é a MedLib?

O projeto consiste em um sistema web inovador desenvolvido pelo Núcleo de Informática do SmartBits (NIS) para o gerenciamento eficiente de livros e materiais escolares, com o objetivo de facilitar o acesso e aprimorar a experiência educacional para estudantes.

Nossos Oferecimentos

Gestão de Livros e Materiais Escolares

O sistema organiza e disponibiliza um catálogo digital de livros e outros recursos escolares, permitindo que os usuários busquem, reservem e acessem os materiais necessários de forma simples e eficiente.

Cursos em Vídeo

Uma biblioteca de cursos em vídeo foi integrada ao sistema, com conteúdos voltados para diversas Tecnologias. Esses cursos foram projetados para atender às necessidades de reforço escolar ou aprendizado independente.

Apoio à Inclusão Tecnológica

O projeto será implementado no laboratório de informática da escola, garantindo que mesmo alunos que não possuem computadores pessoais possam acessar os recursos disponíveis no sistema.

Aprenda Tecnologias Pertinentes

Python

Uma linguagem de programação famosa, simples, versátil. Ideal para desenvolvimento web, automação, análise de dados e muito mais.

JavaScript

A linguagem de programação mais utiliza para desenvolvimento web e também a mais amada por milhares de desenvolvedores

Redes de Computadores

Uma tecnologia essencial para a comunicação e troca de recursos entre dispositivos.

HTML

A principal linguagem de estruturação de código para páginas web. Linguagem de marcação que define a estrutura e o significado do conteúdo da web.
